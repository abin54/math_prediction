# Recursive Language Model (RLM) Genetic Scaffolding

This workflow transforms a standard prediction into a "Genetic Invariant" by analyzing its survival over 52 years.

## Phase 1: The Compressor
Analyze the dataset in 10-year blocks (1972-1982, 1982-1992, 1992-2002, 2002-2012, 2012-2022). For each block, extract the **Dominant Genetic Code** (the rule with the highest success rate):
- Tamil Numerology / Phonetic.
- Vedic Planetary Degrees.
- Pure Math / Binary Step.

## Phase 2: The Evolver
Track how the "Genetic Code" has mutated. Identify rules that have **survived at least three decade-transitions** (30+ years). These are the "Core Invariants."

## Phase 3: The Predictor
Apply the "Evolved Rule" of the current 2022-2026 block to today's date, but only if it matches one of the Core Invariants.
